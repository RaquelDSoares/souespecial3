Programas de Comunicação Aumentativa

Todas estas ferramentas são gratuitas e permitem criar diversos recursos, desde tabelas de rotinas, tabelas de comunicação digitais ou físicas, criar rotinas com tempo limitado, comunicação por voz, adaptação de histórias, criar jogos digitais...
ARASAAC

Nesta plataforma pode encontrar os pictogramas para adaptar os seus materiais. Tem muitos recursos já feitos em português.

Asterics Grid

Software para criar tabelas de comunicação digitais ou físicas. Pode também aqui criar jogos digitais.

Magiccontact

Software para telemóvel e tablet que permite criar tabelas de comunicação e texto para voz, para além de ter acesso à internet, fazer chamadas...

Araword

Esta aplicação permite criar frases ou histórias adaptadas, que depois pode imprimir ou usar em formato digital.

SequenciAAC

Esta aplicação permite criar sequências de pictogramas, para fazer por exemplo sequências de rotinas diárias para imprimir.

Gestiac

O GESTIAC (Gestão do Tempo e da Atividade) permite desenvolver rotinas com tempo definido. A partir da aplicação, podemos configurar cada rotina com um número variável de atividades e definir o tempo que cada uma delas vai durar. 

Picto-selector

Picto Selector é uma aplicação para sistemas operativos Windows que permite realizar placas de comunicação, rotinas, horários, antecipação, mapas conceituais, etc.
